Code Quiz

Click the "Start Quiz" button to begin taking multiple choice quiz. Read the questions, and choose the correct answer for each one. This quiz is timed and scored. You will get a higher score by answering quickly and correctly. TIME: You have 60 seconds to answer all questions. For each question answered incorrectly, 15 seconds will be subtracted from the time remaining. The timer stops when all questions have been answered correctly, or the time remaining equals zero. SCORE: Add 10 points for each correct answer; if all questions are answered correctly before time runs out, add one point for each second left; if time runs out before all questions are answered correctly, subtract five points for each unanswered question. HIGH SCORE: Once the quiz is completed, you will have the option to add your initials and score to the high scores list.
